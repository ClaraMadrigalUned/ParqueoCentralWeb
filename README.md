# Parqueo Central

Aplicación web desarrollada en **ASP.NET MVC 5** sobre **.NET Framework 4.7.2** para la administración de un parqueo. El sistema permite registrar vehículos, gestionar espacios de estacionamiento, controlar las entradas y salidas de los vehículos y calcular automáticamente el monto a cobrar según el tiempo de permanencia.

## Funcionalidades principales

- Registro, consulta, edición y eliminación de vehículos.
- Registro, consulta, edición y eliminación de espacios de estacionamiento.
- Registro de entradas y salidas de vehículos.
- Cálculo automático del monto de cobro (₡600 por hora o fracción).
- Administración del estado de los espacios (Disponible/Ocupado).
- Historial de movimientos.
- Búsquedas dinámicas mediante AJAX.
- Gestión del operador utilizando Session.
- Mensajes de confirmación y error mediante TempData.

## Tecnologías utilizadas

- ASP.NET MVC 5
- .NET Framework 4.7.2
- C#
- Entity Framework 6 (Code First)
- SQL Server LocalDB
- Bootstrap
- JavaScript
- AJAX
- Visual Studio 2022

## Requisitos

- Visual Studio 2022 (o versión compatible con ASP.NET MVC 5)
- .NET Framework 4.7.2
- SQL Server LocalDB

## Instrucciones para ejecutar el proyecto

1. Clonar o descargar este repositorio.
2. Abrir la solución en Visual Studio.
3. Restaurar los paquetes NuGet del proyecto.
4. Compilar la solución.
5. Ejecutar la aplicación utilizando IIS Express.
6. Si la base de datos no existe, Entity Framework la generará automáticamente durante la primera ejecución.
